# Organizar Acciones - PWA

Esta es una Progressive Web App (PWA) diseñada para organizar acciones y decisiones de manera eficiente.

## 🚀 Características Principales
- ✅ Interfaz amigable e intuitiva
- 📱 Funcionalidad offline
- 🔔 Notificaciones push
- 📱 Diseño responsive
- 💾 Almacenamiento local
- ⚡ Rendimiento optimizado
- 🎨 Tema personalizable

## 📥 Instalación

### Clonar el repositorio
```bash
git clone https://github.com/sahamara95-cell/Organizar-acciones-
cd Organizar-acciones-
```

### Usar la PWA
1. Abre la app en tu navegador
2. En navegadores compatibles, verás la opción "Instalar app"
3. Haz clic en instalar para agregar a tu pantalla de inicio
4. ¡Ya puedes usar la app offline!

## 💡 Uso
- **Agregar acciones**: Click en el botón "+" para crear una nueva acción
- **Editar acciones**: Haz clic en una acción para modificarla
- **Eliminar acciones**: Desliza o haz clic en eliminar
- **Offline**: Todas tus acciones se guardan automáticamente

## 🛠️ Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)
- Service Workers
- IndexedDB (almacenamiento local)

## 📋 Estructura del Proyecto
```
Organizar-acciones-/
├── index.html          # Página principal
├── style.css           # Estilos
├── app.js              # Lógica principal
├── service-worker.js   # Service Worker para offline

```
